# Security Policy

Wuxing is a browser game prototype, but security reports are still welcome.

## Supported Versions

The latest commit on `main` is the actively maintained version while the project is pre-release.

## Reporting a Vulnerability

Please do not open a public issue for sensitive security problems.

Use GitHub private vulnerability reporting if it is available for the repository, or contact the maintainer through the repository owner profile.

Helpful details include:

- A clear description of the issue.
- Steps to reproduce.
- Impact and affected files.
- Suggested fix, if known.

The maintainer will review the report and prioritize fixes based on impact.
