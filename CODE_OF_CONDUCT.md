# Code of Conduct

Wuxing aims to be a welcoming open-source project for people interested in web games, cultural mechanics, and TypeScript development.

## Our Standards

Examples of behavior that helps the project:

- Be respectful and constructive.
- Explain technical tradeoffs clearly.
- Welcome beginners and questions.
- Keep discussions focused on the project.
- Give feedback on code, not people.

Examples of behavior that is not acceptable:

- Harassment or personal attacks.
- Dismissive or insulting comments.
- Publishing private information without permission.
- Repeated off-topic disruption.

## Enforcement

Project maintainers may remove comments, close issues, reject contributions, or block participants who do not follow this code of conduct.

If you need to report a concern, open a private security advisory or contact the maintainer through the repository owner profile.
